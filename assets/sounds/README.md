# Sons de l'application Madeen

Ce dossier contient les fichiers audio utilisés dans l'application.

## ✅ Fichiers Installés

- ✅ `success.mp3` - Son de succès (Positive notification)
- ✅ `error.mp3` - Son d'erreur (Alert error notification)
- ✅ `notification.mp3` - Son de notification (Soft bell notification)
- ✅ `click.mp3` - Son de clic (Modern click)

## 📝 Détails

- **Source** : [Mixkit.co](https://mixkit.co/free-sound-effects/)
- **Licence** : Mixkit License (gratuite pour usage commercial et personnel)
- **Format** : MP3
- **Qualité** : Optimisée pour mobile

## 🎵 Utilisation dans l'App

Ces sons sont automatiquement utilisés par le système de sons (`utils/sounds.ts`) :
- `success.mp3` : Quand une habitude est complétée, un dhikr atteint son objectif
- `error.mp3` : En cas d'erreur ou d'action invalide
- `notification.mp3` : Pour les rappels de prière et notifications
- `click.mp3` : Pour les interactions UI (boutons, toggles)

## 🔧 Remplacement

Pour remplacer un son :
1. Gardez le même nom de fichier
2. Format MP3 recommandé
3. Durée < 2 secondes
4. Taille < 100KB
